# ⚙️ Como executar

Clone o projeto

```bash
git clone https://github.com/devPedro17/sistema-odontologico.git
```

Entre na pasta

```bash
cd sistema-odontologico
```

Instale as dependências

```bash
pip install -r requisitos.txt
```

Execute

```bash
python app.py
```

Abra no navegador

```
http://127.0.0.1:5000
```
